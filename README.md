# OrvixHal — Meyve Sebze Toptancı Yönetim Sistemi

**Geliştirici:** Orvix (orvix.co.uk)

## Kurulum (Geliştirici)

```bash
npm install
npm run dev      # Geliştirme modu
npm run build    # Windows .exe oluştur
```

## Teknoloji

- Electron.js v28
- Vanilla JavaScript
- Node.js (crypto, fs)
- bcryptjs, xlsx, electron-builder
